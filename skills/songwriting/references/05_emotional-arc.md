# 05 感情曲線 — 曲は感情の設計図で聴かせる

感情曲線 = 曲頭から曲末までの感情の強度と質の軌跡。歌詞・構成・サウンド演出をこの 1 本に従わせる。ワークフロー段階 1 (設計) で曲線を決め、段階 3 (推敲) でずれを検査する。

## 1. 曲線の書き方 (設計の成果物)

各セクションに**感情強度 1〜10** と**感情の質 (1 語)** を割り付け、構成表 (01 §5) に記入する。

例 (V 字型。自作の想定曲「夜光性メロディ」):

| セクション | 強度 | 質 |
|---|---|---|
| `[Intro]` | 2 | 予感 |
| `[Verse 1]` | 3 | 孤独 |
| `[Pre-Chorus]` | 5 | 焦り |
| `[Chorus]` | 8 | 切望 |
| `[Verse 2]` | 4 | 回想 |
| `[Pre-Chorus]` | 6 | 決意の芽 |
| `[Chorus]` | 8 | 切望 |
| `[Bridge]` | 3 | 静けさ (どん底) |
| `[Chorus]` (大サビ) | 10 | 解放 |
| `[Outro]` | 4 | 余韻 |

ルール:

- **強度 10 は 1 曲に 1 箇所だけ** (たいてい大サビ)。2 箇所あるとどちらも 10 に聞こえない。
- 同じ強度が 3 セクション続いたら平坦。どこかを上げ下げする。
- 隣接セクションの強度差 5 超の跳躍は 1 曲 1 回まで (Bridge → 大サビの専売)。

「質」の語彙が出ないときは次から選ぶ: 予感 / 孤独 / 焦り / 憧れ / 切望 / 回想 / 決意 / 静けさ / 強がり / 安堵 / 解放 / 祝祭 / 余韻。

## 2. 基本パターンと選び方

| パターン | 形 | 合う題材 |
|---|---|---|
| **上昇型** | 3 → 5 → 7 → 10 と積み上げる | 応援歌、旅立ち、始まりの曲 |
| **V 字型** | 中盤 (Bridge) で一度落としてから最大へ | 喪失からの再生、葛藤もの。最も汎用 |
| **静→爆発型** | 前半を 2〜4 で抑制し、後半で一気に解放 | 抑圧からの解放、告白もの |
| **余韻型 (下降)** | ピークを 2 回目サビに置き、後半は静かに閉じる | 別れ、追憶、AABA 型バラード |

選び方: ブリーフに感情曲線の指定があれば従う。なければ題材で選び、迷ったら V 字型。

## 3. 1 番と 2 番の情報差

2 番が 1 番の言い換えだけの曲は、2 番で聴くのをやめられる。**2 番には新情報を 2 つ以上**入れる。

情報差の作り方 (どれか 1 つを軸にする):

| 軸 | 1 番 | 2 番 |
|---|---|---|
| 時間経過 | 出会った夜 | 三ヶ月後の朝 |
| 視点転換 | 私から見た君 | 君から見えていたはずの私 |
| 内面 → 外面 | 心の中の独白 | 実際に交わした会話・行動 |
| 抽象 → 具体 | 予感・気配 | 決定的な出来事 |

- 検査法: **2 番だけを読んで、1 番にない事実・場面・転回を 2 つ挙げられるか**。
- Suno 特性: `[Verse 1]` / `[Verse 2]` の番号付きタグは別のメロディ展開を誘導する (01 §3)。歌詞側も別内容にするのが噛み合う。

実例 (時間経過軸。自作)。1 番 A メロの抜粋:

```text
改札で 手を振った
帰り道は まだ二人分
```

2 番 A メロの抜粋:

```text
おなじ改札 通り抜ける
定期券だけ 更新されてく
```

同じ場所 (改札) を再訪させると時間経過が 1 行で伝わり、モチーフの回収 (04 §2) も兼ねられる。新情報は「一人になった」「季節が進んだ」の 2 つ。

## 4. ブリッジ → 大サビのカタルシス設計

カタルシスは「落差」で作る。手順:

1. **Bridge で曲中最低クラスの強度まで一度落とす** (V 字型なら 2〜3)。音数も減らす: 行を短く・少なく (2〜4 行)
2. Bridge の内容は次のどれか 1 つに絞る: (a) 最も個人的な告白、(b) 時間・視点の飛躍、(c) それまでの前提の反転 (04 §2)
3. 落ちサビを挟むなら、同一のサビ歌詞を `[Chorus: stripped down, quiet vocals]` で静かに歌わせ、続く大サビとの落差を最大化する
4. **大サビで最大強度**。ここだけに許される演出: `[Key Change]`、サビ歌詞の 1 語差し替え (§5)

## 5. 大サビでの歌詞変化 (1 語差し替えの技法)

Suno は同一歌詞の `[Chorus]` 反復でメロディを固定する (01 §3)。大サビで歌詞を大きく変えるとメロディごと変わるリスクがある。両立の妥協点:

- **サビ歌詞の 1 箇所 (1 語〜1 行) だけを差し替え、残りは一字一句同じにする**
- 差し替え位置はサビ末行が安全 (頭を変えるとフックが揺れる。02 §2)
- 差し替えの定石: 否定 → 肯定 / 疑問 → 断言 / 過去 → 現在
- 例 (自作): 通常サビ末「まだ言えないまま」→ 大サビ末「もう言える気がした」(モーラ数はほぼ維持する)

### 同一歌詞のまま感情を変える 2 つの道具

サビ歌詞を固定しても (01 §3)、聞こえ方は変えられる。

1. **直前の行で文脈を変える** — 2 番サビの直前 (Pre-Chorus 末) に反転の 1 行を置くと、同じサビが別の意味で響く (1 番では願望だった行が、2 番では強がりに聞こえる、など)
2. **演出タグで歌わせ方を変える** — 落ちサビ `[Chorus: restrained, almost whisper]` → 大サビ `[Chorus: full band, soaring vocals]`。歌詞は同一のまま強度だけを動かせる (§6)

## 6. 感情とサウンドの連動 (パラメータ付きタグへの翻訳)

感情曲線は歌詞だけでなく音にも指示する。**強度と質をタグ修飾子に翻訳**し、該当セクションのタグに付ける (タグ本体は suno-spec の語彙、修飾子は短い英語の形容)。

| 曲線上の状態 | タグ修飾の例 |
|---|---|
| 低強度・内省 (2〜3) | `[Verse 1: soft vocals, minimal arrangement]` |
| 傾斜・緊張 (5〜6) | `[Pre-Chorus: building intensity]` |
| ピーク (8) | `[Chorus: full band, soaring vocals]` |
| どん底・無防備 (Bridge) | `[Bridge: stripped down, piano only, vulnerable vocals]` |
| 最大 (10) | `[Chorus: explosive, layered harmonies]` |
| 余韻 | `[Outro: fading, gentle]` |

- 修飾子は 1 タグに 1〜3 個まで。**全セクションに付けると効きが薄まる** — 曲線の要所 (どん底・最大・転換点) だけに付ける。
- composer も同じブリーフの感情曲線から Style と演出を設計する。作詞家のタグ修飾は「セクション単位の感情演出」に限り、ジャンル・楽器編成の全体指定 (Style 欄の領分) を書かない。

## 7. 推敲チェックリスト

- [ ] 強度 10 が 1 箇所だけにある
- [ ] 同じ強度が 3 セクション以上続いていない
- [ ] 2 番だけ読んで、新情報を 2 つ挙げられる
- [ ] Bridge が「落とす」役割を果たしている (行数・音数も減っている)
- [ ] 大サビの歌詞変化が 1 箇所以内に収まっている (Suno のメロディ固定と両立)
- [ ] タグ修飾子が要所だけに付いており、Style 欄の領分を侵していない
- [ ] 各セクションの歌詞のトーンが、割り付けた「感情の質」と一致している
